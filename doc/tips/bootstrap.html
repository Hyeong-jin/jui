Mouse Right-Click (+Resize)

로우를 마우스 오른쪽 클릭하면 rowmenu 이벤트가 발생합니다.
이 이벤트를 활용하면 로우를 컨트롤할 수 있는 메뉴 등을 구현하는데 활용할 수 있습니다.

Sorting (+Resize)

정렬 기능을 사용하기 위해서는 th 태그 순서대로 데이터 키 값을 넣어주고, 정렬 대상이 아닐 경우에는 null로 대체하면 됩니다.
또한 정렬 중임을 보여주는 UI는 별도로 구현할 수 있도록 관련된 인터페이스를 제공합니다.

Edit Row (+Sorting)

특정 로우를 더블클릭하면 모든 셀의 값을 수정할 수 있습니다.
참고로 rowEdit 옵션의 값이 true가 아니고, 컬럼 인덱스 배열이라면 목록의 컬럼들만 수정할 수 있습니다.
